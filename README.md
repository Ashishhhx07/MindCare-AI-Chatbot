# MindCare AI Chatbot

MindCare AI is a mental health support chatbot that helps users express emotions and receive basic emotional guidance.

This project, **MindCare AI Chatbot**, was developed as a prototype for a hackathon to demonstrate how AI-powered conversational systems can support mental well-being by detecting mood and providing helpful suggestions such as breathing exercises.

I enjoy exploring new technologies and continuously improving my programming and problem-solving skills.

## Features
- AI chatbot interaction
- Mood detection
- Breathing exercise suggestion
- Modern UI with animations
- Stress support responses

## Tech Stack
- Python (Flask)
- HTML
- CSS
- JavaScript

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Run server
python app.py

3. Open browser
http://127.0.0.1:5000
